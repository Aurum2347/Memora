# Memora

Memora - это приложение для заметок, где каждая запись оформлена в стиле поста. Оно создано для тех, кто хочет быстро фиксировать мысли, идеи и задачи без лишнего визуального шума.

## Основные возможности

- **Пост-формат**: Заметки выглядят как лента постов с датой, тегами и содержимым.
- **Черновики**: Возможность сохранять незавершенные записи отдельно от основной библиотеки.
- **Теги и поиск**: Быстрая навигация по записям с помощью тегов и полнотекстового поиска.
- **Календарь**: Визуальное отображение дней, когда вы делали записи.
- **Локальное хранение**: Все данные хранятся на вашем компьютере в обычных текстовых файлах (.md). Никаких облаков и регистрации.
- **Граф связей**: Визуализация взаимосвязей между вашими заметками.
- **Темная тема**: Интерфейс адаптирован для комфортной работы в любое время суток.

## Установка и запуск

1. Скачайте последнюю версию приложения из раздела Releases.
2. Запустите скачанный файл `Memora.exe`. Установка не требуется.
3. При первом запуске выберите папку, где будут храниться ваши заметки. Это может быть любая папка на компьютере или внешний диск.

## Хранение данных

Все ваши заметки - это обычные Markdown-файлы. Вы можете открыть их любым текстовым редактором, скопировать на флешку или синхронизировать через Dropbox/Google Drive. Приложение создает индексный файл `memora-index.json` в выбранной папке для быстрой загрузки списка заметок.

## Системные требования

- Windows 10 / 11 (x64)
- Свободное место на диске: ~200 МБ для самого приложения
